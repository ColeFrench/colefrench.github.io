Userscripts

Userscripts are JavaScript files which are automatically run when a certain webpage is visited (or on all webpages, depending on configuration). Their purpose is to add extra features to webpages (e.g., aesthetics, functionality).

They require Greasemonkey to run on Mozilla Firefox and Tampermonkey to run on Google Chrome.
